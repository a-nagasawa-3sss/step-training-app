タスク1: VPC リソースの作成

タスクの GOAL

  • 必要な VPC リソースを作成する
  • サービスを起動し、確認を行った後は、必ず都度サービスを停止させる

タスクの流れ

  1. VPC とサブネットを作成する
  2. EC2 用のセキュリティグループを作成する
  3. RDS 用のセキュリティグループを作成する
  4. RDS 用のサブネットグループを作成する

タスクの要件

VPC：名称 wordpress-vpc / CIDR 10.0.0.0/16 / リージョン ap-northeast-1

サブネット：Public 1a(10.0.1.0/24) / Public 1c(10.0.2.0/24) / Private 1a(10.0.3.0/24) / Private 1c(10.0.4.0/24)。Public はIGWへの経路あり、Privateはなし。

セキュリティグループ：ec2-sg(HTTP 80を0.0.0.0/0、SSH 22) / rds-sg(MySQL 3306をec2-sgから)

RDS用サブネットグループ：wordpress-db-subnet（2つのプライベートサブネット）

チェックポイント

VPC一覧・サブネット一覧・セキュリティグループ一覧・DBサブネットグループ画面のスクリーンショットをアップロードしてください（複数枚アップロード可）。
